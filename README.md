# Install-ROS2-Foxy-Fitzroy-and-Gazebo-11-in-Ubuntu-20.04-with-VS-Code

Ubuntu 20.04에서 VS code, ROS2 Foxy, Gazebo 11을 쉽게 다운로드할 수 있도록 제작하였다.

아래 코드를 입력하면 쉽게 다운할 수 있다.

```javascript
$ wget https://raw.githubusercontent.com/knowledge-intelligence/ROS_RobotManipulator/master/ros2_foxy_install.sh && chmod 755 ./ros2_foxy_install.sh && ./ros2_foxy_install.sh
```
